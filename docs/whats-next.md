# What's next

[‹ Overview](README.md)

ProcessKit is not one library but a family: the same approach — kernel-backed
whole-tree kill-on-dispose, honest results (a non-zero exit is *data*, not a
raised error, until you ask for success), and testable seams — brought to
several ecosystems, each in its own repository and documented on its own site.

This site documents the **.NET** implementation, published as
[ProcessKit](https://www.nuget.org/packages/ProcessKit) on NuGet and designed
for first-class, idiomatic use from **both F# and C#**. Its sibling, the
original **Rust** crate, is already published; **Go** and **Kotlin**
implementations are planned, and a **Python** wrapper will follow. Every
implementation follows the same philosophy and is documented alongside this one
as it ships — the reserved menu entries below link out to the ones that are
live.
